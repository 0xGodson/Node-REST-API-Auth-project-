Node Application

REST API (Node JS)

Create Account, Store passwords in DB as Hash, Login

To Create account, send a post req. to /api/register. ex: content-type: Application/json { "username": "godsod", "password": "pass" }

To Login. Send a Post req. to /api/login. ex: content-type: Application/json { "username": "godsod", "password": "pass" }
